Link events and posts to team members who collaborate on them.

== Description ==

Fair Team allows you to link events, posts, and other content to team members who collaborate on them. This makes it easy to showcase who worked on an event or project.

**Features:**
* Link team members to events and posts
* Display team member information
* Manage team collaborations
* Integration with Fair Event Plugins ecosystem

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/fair-team/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Configure settings via the admin menu
4. Start linking team members to your events and posts

== Frequently Asked Questions ==

= What content types can I link team members to? =

You can link team members to events (if using Fair Events plugin) and any WordPress posts or custom post types.

= Does this require other Fair Event plugins? =

No, Fair Team works standalone. However, it integrates seamlessly with other Fair Event plugins like Fair Events and Fair RSVP.
