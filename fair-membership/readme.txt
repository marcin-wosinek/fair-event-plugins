Membership management plugin.

== Description ==

Fair Membership is a WordPress plugin designed for managing group memberships.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/fair-membership` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
