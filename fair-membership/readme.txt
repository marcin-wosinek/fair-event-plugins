Membership management plugin for events, with fair pricing model.

== Description ==

Fair Membership is a WordPress plugin designed for managing memberships for events with a focus on fair pricing models.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/fair-membership` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
